    Solusi Numeri Persamaan Differensial Biasa¶

    Persamaan Differensial merupakan sebuah persamaan yang terdiri dari beberapa turunan biasa dari fungsi yang tidak diketahui dari variabel bebas tunggal dan variabel tersebut hanya disimbolkan degna x. Berikut ini beberapa contoh differensial biasa.

    Variabel bebas dari contoh tersebut adalah dan variabel terikatnya adalah yang merupakan fungsi dari x atau juga bisa ditulis y = f(x). Persamaan Differensial Biasa dikelompok kan berdasarkan oredenya yaitu:

    1. Persamaan Differensial orde 1 dimana turunan tertingginya adalah turunan pertama, pada contoh diatas berada pada nomor 1,2 dan 3/
    2. Persamaan Differensial orde 2 dimana turunan tertingginya adalah turunan kedua, pada contoh diatas berada pada nomor 4.
    3. Persamaan Differensian Biasa orde 3 dimana turunan tertingginya adalah turunan ketiga, pada contoh ditas berada pada nomor 5.

    Bentuk baku persamaan differensial biasa dapat ditulis seperti berikut: dengan niilai awal y(x0) = y

    Persamaan differensial biasa orede satu tidaka mengikuti bentuk umum tersebut jadi harus ditulis ulang agar dapat diselesaikan. Penyelesaian persamaan differensial biasa secara numerik berarti menghitung nilai fungsi di , dengan h merupakan ukuran langkah setiap lelaran. pada metode analitik nilai awal digunakan untuk merperoleh solusi yang unik , sedangkan pada metode numerik nilai awal berfungsi untuk memulai lelaran.

    Dibawah ini akan dijelaskan beberapa solusi numerik untuk menyelesaikan persamaan differensial biasa.

    A. Metode Eular¶

    Metode eular adalah metode untuk menyelesaikan persamaan differensial biasa yang dihasilkan daru turunan deret Tylor. Metode eular juga disebut sebagai metode orde pertama karena hanya mengambil sampai suku pertama saja dari deret Tylor. Metode eular juga dapat di turunkan dengan menggunakan metode aturan segi empat untuk menginsterasikan pada persamaan differensial. bentuk metode eular sebagai berikut: Contoh selesaikan persamaan differensial berikut:

    Diketahui persamaan differensial biasa sebagai berikut gunakan metode eular untuk menghitung y(0,10) dengan ukuran langkah h = 0.02 dan diketahui solusi persamaan differensial biasa sejatinya = 1,1103

    2. Algoritma Metode Eular¶

    1. Deklarasikan fungsi persamaan differensial biasa yang akan di selesaikan.

    2. Masukkan inputan yang dibutuhkan seperti x(0), y(0), xb dan h

    3. Hitung perulangan yang diperlukan dengan menggunakan rumus

    4. Inisialisasi nilai x dan y awal sama dengan 0 dan y(0).

    5. Tentukan nilai analitiknya untuk menghitung nilai eror
    6. lakukan perulangan sebanyak n kali.
    7. Setiap kali melakukan perulangan maka nilai x dan y akan diupdate dengan menggunakn persamaan metode Eular, dan hitung nilai erornya kemudian tampilkan hasilnya
    3. Code metode Eular¶

    berikut ini merupakan code hasil dari implementasi algoritma di atas:

    def f(x,y):
    return x + y
print("Penyelesaian Persamaan Differensial Biasa Menggunakan Metode Euler")
x0 = float(input("Masukkan Nilai X0 : "))
y0 = float(input("Masukkan Nilai y0 : "))
xb = float(input("Masukkan Nilai X yang dicari : "))
h = float(input("Masukkan Nilai h : "))
n = (xb - x0)/h
iterasi = int(n)
x = x0
y = y0
true =  1.1103
e = abs(true-y)
print ("==================================================")
print (">>>>>>>>>>>>>>>>>>- METODE EULER -<<<<<<<<<<<<<<<<")
print ("==================================================")
print ('|i','\t|','x(i)','\t\t|','y(i)\t\t|e')
print ("|%d"%x0,'\t|',"%f"%x,'\t|',"%f\t|"%y,"%f"%e)
for i in range (iterasi):
    y = y + h*f(x,y)
    x = x + h
    e = abs(true-y)
    print ("|%d"%(i+1),'\t|',"%f"%x,'\t|',"%f\t|"%y,"%f"%e)
print ("==================================================")

    B. Metode Heun¶

    Metode Heun adalah metode perbaikan dari metode eular yang memiliki nilai eror yang tinggi. Pada metode ini solusi dari metode Eular dijadikan sebagai solusi perkiraan awal (predictor), lalu solusi perkiraan awal tersebut diperbaiki dengan metode Heun (corecor). Sehingga persamaannya dapat ditulis sabagai berikut: Contoh Soal:

    Diketahui PDB dan gunakan metode Heun untuk menghitung dengan ukuran langkah . Dan diketahui solusi PDB sejatinya =

    1. Algoritma Metode Heun¶
    1. Deklarasikan fungsi persamaan differensial biasa yang akan diselesaikan.
    2. Masukkan inputan seperti , dan .
    3. Hitung dan buat perulangan yang diperlukan dengan menggunakan rumus ,
    4. Inisialisasi nilai dan awal sama dengan dan ,
    5. Tentukan nilai analitiknya untuk menghitung nilai eror.
    6. Lakukan perulangan sebanyak nilai dari n.
    7. Setiap melakukan perulangan maka nilai predictor dan corrector serta akan diupdate dengan menggunakan persmaan metode Heeun, lalu hitung nilai erornya dan tampilkan hasilnya.
    2. Code Metode Heun¶

    berikut ini merupakan code hasil dari implementasi algoritma di atas.

    def f(x,y):
    return x + y
print("Penyelesaian Persamaan Differensial Biasa Menggunakan Metode Heun")
x0 = float(input("Masukkan Nilai X0 : "))
y0 = float(input("Masukkan Nilai y0 : "))
xb = float(input("Masukkan Nilai X yang dicari : "))
h = float(input("Masukkan Nilai h : "))
iterasi = (xb - x0)/h
n = int(iterasi)
x = x0
y = y0
true =  1.1103
e = abs(true-y)
print ("==================================================")
print (">>>>>>>>>>>>>>>>>>- METODE HEUN -<<<<<<<<<<<<<<<<<")
print ("==================================================")
print ('|i','\t|','x(i)','\t\t|','y(i)\t\t|e')
print ("|%d"%x0,'\t|',"%f"%x,'\t|',"%f\t|"%y,"%f"%e)
for i in range (n):
    ypre = y + h*f(x,y)
    y = y + h/2*(f(x,y)+f(x+h,ypre))
    x = x + h
    e = abs(true-y)
    print ("|%d"%(i+1),'\t|',"%f"%x,'\t|',"%f\t|"%y,"%f"%e)
print ("==================================================")

    3. Metode Runge-Kutta¶

    Pada metode Runge-Kutta persamaan differensial dapat diselesakan dengan ketelitian yang lebih tinggi, dan menghindarin mencari turunan yang lebih tinggi dengan cara mengevaluasi fungsi pada titik terpilih dalam setiap langkah. Metode ini paling banyak digunakan dalam praktek.

    bentuk umum metode runge-kutta orde-n adalah sebagai berikut: dengan adalah tetapan dan Dimana nilai dipilih sedemikian rupa sehingga meminimumkan galat perlangkah. Berikut ini jenis jenis metode Runge-Kutta berdasarkan orde nya.

    1. Metode Runge-Kutta Orde 1¶

    Metode ini berbentuk Salah satu bentuk contoh metode Runge-kutta orde 1 adalah memiliki bentuk persamaan seperti metode Euler.

    contoh soal:

    Diketahui PDB dan gunakan metode Runge-Kutta orde 1 untuk menghitung dengan ukuran langkah . Dan diketahui solusi PDB sejatinya =

    a. Algoritma Runge-Kutta Orde 1¶
    1. Deklarasikan fungsi persamaan differensial nya
    2. Masukkan inputan seperti , dan .
    3. Hitung dan buat perulangan yang diperlukan dengan menggunakan rumus .
    4. Inisialisasi nilai dan awal sama dengan dan ,
    5. Tentukan nilai analitiknya untuk menghitung nilai eror.
    6. lakukan perulangan sebanyak n.
    7. nilai , nilai dan nilai serta nilai akan diupdate setiap kali melakukan perulangan, lalu cetak nilai nilai tersebut setiap melakukan perulangan.

    2. Metode Runge-Kutta Orde 2¶

    Metode Orde 2 berbentuk Salah satu bentuk contoh metode Rengu-Kutta orde 2 dapat dilihat dari bentuk persamaanya yang hampir mirip seperti metode Heun.

    Contoh Soal

    Diketahui PDB dan gunakan metode Runge-Kutta orde 2 untuk menghitung dengan ukuran langkah . Dan diketahui solusi PDB sejatinya =

    a. Algoritma Metode Runge-Kutta Orde 2¶
    1. Deklarasikan persamaan differensial biasanya
    2. Masukkan inputan seperti , dan .
    3. Hitung dan buat perulangan yang dibutuhkan dengan menggunakan rumus .
    4. Inisialisasi nilai dan awal sama dengan dan .
    5. Tentukan nilai anlitiknya untuk digunakan menghitung eror.
    6. Lakukan perulangan sebanyak n.
    7. nilai dan , nilai dan nilai serta nilai akan di update dan di cetak setiap kali melakukan perulangan.
    b. Code Metode Runge-Kutta Orde 2¶

    Berikut ini merukapakan code hasil dari implementasi algoritma di atas.

    def f(x,y):
    return x+y
print("Penyelesaian Sistem Persamaan Differensial Biasa Menggunakan Metode Runge-Kutta Orde2")
x0 = float(input("Masukkan Nilai X0 : "))
y0 = float(input("Masukkan Nilai y0 : "))
xb = float(input("Masukkan Nilai X yang dicari : "))
h = float(input("Masukkan Nilai h : "))
iterasi = (xb - x0)/h
n = int(iterasi)
x = x0
y = y0
true =  1.1103
e = abs(true-y)
print ("==================================================================================")
print (">>>>>>>>>>>>>>>>>>>>>>>>>>>>>- METODE RUNGE-KUTTA2 -<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<")
print ("==================================================================================")
print ('|i','\t|','x(i)','\t\t|','y(i)\t\t|e\t\t|k1\t\t|k2')
print ("|%d"%x0,'\t|',"%f"%x,'\t|',"%f\t|"%y,"%f"%e,"\t|0\t\t|0")
for i in range (n):
    k1 = h*f(x,y)
    k2 = h*f(x+h,y+k1)
    y = y +(k1 +k2)/2
    x = x+h
    e = abs(true-y)
    print ("|%d"%(i+1),'\t|',"%f"%x,'\t|',"%f\t|"%y,"%f\t|"%e,"%f\t|"%k1,"%f"%k2)
print ("==================================================================================")
